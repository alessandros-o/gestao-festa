# gestao-festa

## API para gestão de convidados e acompanhantes

### Tecnologias utilizadas
* Linguagem JAVA
* Spring MVC
* Spring Boot
* Spring Data JPA
* Spring Security
* Thymeleaf
* Banco de dados MySQL
* 

### Funcionalidades
* Cadastro de convidados com a quantidade de acompanhantes
* Restrição de segurança com acesso por senha
